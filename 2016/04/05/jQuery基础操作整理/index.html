jQuery基础操作整理

发表于 2016-04-05 
这是根据慕课网的jQuery课程整理的文章。

jQuery基础选择器

jQuery的选择器都是以$符号开头，基本和querySelector选择器很像。
  1. $(‘#id’)
  2. $(“element”)
  3. $(“.class”)
  4. $(*)
  5. $(“sele1, sele2, seleN”)
  6. $(“ance desc”)
  7. $(“parent > child”)
  8. $(“prev + next”)
  9. $(“prev ~ sibling”)

jQuery过滤性选择器

类似于伪类的方法
  1. $(“:first”)
  2. $(“:eq(index)”) // start at 0
  3. $(“:contains(‘text’)”)
  4. $(“:has(selector)”)
  5. $(“:hidden”)
  6. $(“:visible”)
  7. $(“[attribute=value]”)
  8. $(“[attribute!=value]”)
  9. $(“[attribute*=value]”)
  10. $(“:first-child”)
  11. $(“:last-child”)

jQuery表单选择器

全部基于表单
  1. $(“ :input”) // 获取全部表单元素，注意前面有空格
  2. $(“ :text”) // 获取单行文本输入框元素
  3. $(“ :password”)
  4. $(“ :radio”)
  5. $(“ :checkbox”)
  6. $(“ :submit”) // 提取对象：button按钮以及type=submit的input
  7. $(“ :image”) // 只能提取到type为image的input
  8. $(“ :button”) // 提取对象：普通button按钮以及type=button的input
  9. $(“ :checked”) // 与元素之间没有空格 如$("form input:checked")
  10. $(“ :selected”) // as upper

jQuery操作DOM元素

  1. 使用attr()方法控制元素属性
    attr(属性名， (属性值))
  2. 使用 html() 和 text() 方法操作元素内容
    html()可以获取原文的格式代码（如同innerHTML）

  3. 使用 addClass() 和 css() 可以方便地操作元素样式

  4. 使用removeAttr(name) 和 removeClass(class)分别可以实现移除元素的属性和样式的功能

  5. 使用append()方法向元素内追加内容
  6. 使用appendTo()方法向被选元素内插入内容
    $(content).appendTo(selector)
  7. 使用before() 和 after()在元素前后插入内容
    $(selector).before(content)
$(selector).after(content)
  8. 使用clone()方法复制元素
    $(selector).clone()

  9. 使用replaceWith() 和 replaceAll()方法替换元素或元素中的内容，但二者的位置不同

  10. 使用wrap() 和 wrapInner()方法包裹元素和内容，前者用于包裹元素本身，后者用于包裹元素中的内容

    $(selector).wrap(wrapper)
$(selector).wrapInner(wrapper)
  11. 使用each()方法遍历元素
    $(selector).each(function(index))
  12. 使用remove() 和 empty()方法删除元素，remove()方法删除所选元素本身和子元素，可通过过滤参数指定要删除的元素，而empty()方法只能删除所选元素的子元素
